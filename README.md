# Tic-Tac-Toe-Python

A 3x3 tic tac toe game

# How to play

You start a game by:

```python
python game.py
```

![Alt text](tic_tac_toe_1.png?raw=true "GUI")

Click a button to start, then you will play with AI

You can change AI with Dumb computer in game.py
